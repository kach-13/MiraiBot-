给使用MCL整合包用户的提示。

你需要将旧整合包目录里的以下文件夹移到本整合包目录下，
启动整合包才会有旧整合包的功能

bots (账号配置，如果旧整合包登录不了了不要复制)
config (插件配置文件)
data (插件数据)
plugins (插件本体)

除非你知道那些文件和目录是插件运行所必须的，否则不要复制。

更新一键包.bat 最多只能更新到测试版，本包多数开发版，一般用不到这个bat

帮助文档 (新): https://wiki.mrxiaom.top/mirai
帮助文档 (旧): https://mirai-docs.doomteam.fun/

本一键包 Mirai 版本为 2.15.0-dev-98，加密算法SPI更新，支持扫码登录
扫码登录命令【qrlogin QQ号 协议】，可不填协议，默认手表
扫码登录要配置自动登录请使用 /qrAutoLogin 命令
更多扫码登录相关帮助，详见 https://mirai.mamoe.net/topic/2134

你可以在 /config/xyz.cssxsh.mirai.fix-protocol-version/ 文件夹添加文件来修改协议信息，
详见 https://github.com/MrXiaoM/fix-protocol-version
不要乱改，自担风险。

关于 tlv544 加密算法添加示例请见 https://github.com/MrXiaoM/KawaiiMiku

要运行 mirai，打开【Start.bat】即可，包内已自带 zulu jre 11.0.14 x64 并且我们已经优化好了启动脚本，你不需要担心 java 的问题

Linux 或 MacOS 应自己配java并删除本包的java文件夹，运行【./mirai】或【./Start】，
主要是塞三份java会让一键包非常大，所以就不给这两个系统配好java了。
据说有群友打不开 ./mirai，于是我写了另一个文件头不同的 sh 脚本，总之这两个脚本哪个能用就用哪个。
不推荐使用 root 用户运行，权限不足时使用【chmod 777 ./Start】授予权限

如果有疑问可以到群【1047497524】带图询问

打开mirai后，使用命令【login 账号 密码】来登录，注意有空格。尽量不要用新注册的QQ号
登录视频教程: https://www.bilibili.com/video/BV1eb4y1k7uX/

如果你想要自动登录，用 VSCode 之类的文本编辑器打开 config/Console/AutoLogin.yml，里面有详细的注释了，不多解释
扫码登录的自动登录和密码登录的自动登录是不同的，需要注意。

无法登录的常见原因:
格式为:消息	可能的原因	可能的解决方案

密码错误	密码错误或过长	手机协议最大支持 16 位密码. 在官方 PC 客户端登录后修改密码
code=45 	协议版本过低或设备信息被拉黑	删除 device.json, 让其重新生成
code=235	协议版本过低或设备信息被拉黑	删除 device.json, 让其重新生成
code=237	滑块验证处理过慢或者提交ticket有误	尝试使用 https://github.com/KasukuSakura/mirai-login-solver-sakura 处理滑块验证
code=238	当前协议已禁止密码登录	使用扫码登录
